# tdx-go
tdx行情go版本
参考pytdx,使用golang实现
# 提示
- 检测最优路线时需使用sudo权限
# fork方法
- cd $GOPATH/src/github.com/cyclegen
    进入cyclegen工作目录下
- git clone git@github.com:cyclegen/tdx-go.git
    clone主线仓库
- cd $GOPATH/src/github.com/cyclegen/tdx-go
    进入tdx-go项目
- git remote add fork github.com/me/tdx-go.git
    添加远程仓库
- git push fork
    推送到远程仓库
